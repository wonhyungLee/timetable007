# Changelog

## 1.2.0 - 2026-02-27

### Added
- 전담(special) 수업 **교환(수업↔수업)** 시에도, 교사 동시간대 중복/상대 전담의 원래 시간 중복으로 교환이 막히는 경우 **자동 해결안(Plan) 제안**
  - 교환을 먼저 가정한 뒤, 양쪽 전담 충돌을 각각 해결하는 플랜을 결합해 **2~3단계 조치**를 제안
  - 적용 후 **Undo 1회**로 원복 가능
  - 해결이 어려운 경우 모달 내 **강제 교환(빨간 테두리 표시)** 옵션 제공

### Notes / Limitations
- 자동 해결안은 전담 교사 충돌(동시간대 중복) 중심으로 탐색합니다.
- 휴업일/교사 담당 학급 불일치 등은 구조적으로 해결이 어려울 수 있으며, 이 경우 강제 교환 또는 다른 시간대를 선택하세요.

## 1.1.0 - 2026-02-27

### Added
- 전담(special) 수업 **이동(빈칸 이동)** 시, 교사 동시간대 중복으로 이동이 막히는 경우 **자동 해결안(Plan) 제안**
  - 다른 학급의 전담 수업을 먼저 이동/교환한 뒤 다시 이동하는 **2~3단계 조치**를 한 번에 적용
  - 적용 후 **Undo 1회**로 원복 가능

### Notes / Limitations
- 현재 자동 해결안은 **전담 수업 + 이동 모드(대상 칸이 빈칸)** 에 우선 적용됩니다.
- 휴업일 칸/교사 담당 학급 불일치 등의 경우에는 자동 해결안으로 해결이 어려울 수 있으며, 다른 칸을 선택하거나(혹은) 강제 이동을 사용할 수 있습니다.
